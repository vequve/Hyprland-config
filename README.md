## How to apply

First of all you need Hyprland, waybar and hyprpaper.

Then:
```
git clone https://github.com/yazy0/hyprland-conf
cd hyprland-conf
mv .config/hypr
mv .config/waybar
mv .config/kitty
```

And that's about it!
